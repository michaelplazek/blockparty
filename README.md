# Cash For Cypto

## Setup
Install dependencies
```
yarn
```

(Optional) Set port
```
export PORT=3000
```

Set Google Maps API Key
```
export GOOGLE_MAPS_KEY=<API key>
```

Set the Blocktap API Key
```
export BLOCKTAP_TOKEN=<API key>
```

Start the [API](https://github.com/michaelplazek/cash-for-crypto-api)
then run `npm start`
